# Identification of 2'OH hydrogen bonds in translation pre-initiation complexes 

The following structures were downloaded from the Protein Data Bank: PDB:7UCJ, PDB: 6YAL, PDB: 8PJ1, PDB: 8PJ2, PDB: 8PJ3, PDB: 8PJ4, PDB: 8PJ5, PDB: 8PJ6. Structural visualization and measurements were performed in PyMOL using the code from the *.pml files.
